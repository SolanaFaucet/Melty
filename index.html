Melty v0.2.0

AI chat for power users.

Write code, interact with your file system, browse the web, and easily chat with different language models on your desktop. All in a blazing fast native app.

A different kind of chat app

Multiplayer Chat

Instead of just one AI, Melty conversations can have lots of participants, all with different abilities. Hover over their names to see what they do.

Powerful tools out of the box

Melty can search the web, edit code, save memories, generate images, and run terminal commands.

Versioned with git

Branch, revert, and checkout any previous state. Under the hood, each message is a git commit. See a live graph of your chat history.

Built for power users

Instant full-text search across your chats. Keyboard shortcuts for everything. Blazing fast native Mac app.

Interact with your file system

Melty can open and edit files and run terminal commands. Each chat is saved to your local file system.
